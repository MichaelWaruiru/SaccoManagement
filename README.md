- ## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/MichaelWaruiru/SaccoManagement.git
    ```
2. Navigate to the project directory:
    ```bash
    cd SaccoManagement
    ```
3. Install dependencies:
    ```bash
    go get ./...
    ```
4. Build the project:
    ```bash
    go build
    ```
